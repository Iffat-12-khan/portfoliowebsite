# Portfolio Website

A modern and responsive personal portfolio website built with React. This project showcases your profile, skills, projects, and contact information in a clean one-page layout.

## Features

- Responsive navigation bar
- Home section with introduction
- About section
- Skills section
- Projects section
- Contact section
- Footer section
- Smooth section navigation through anchor links

## Tech Stack

- React
- React DOM
- Create React App
- CSS for styling

## Project Structure

- src/App.js - Main app layout and section structure
- src/components/ - Reusable UI sections such as Navbar, Home, About, Skills, Projects, Contact, and Footer
- src/App.css - Global styling for the app
- public/ - Static assets and HTML template
